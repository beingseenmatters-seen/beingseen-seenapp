# Seen App Prototype

A minimalist, mobile-first prototype for the "Seen" application.

## Core Features
- **Reflect**: Expression and AI mirroring.
- **Resonate**: Frequency-based connection discovery.
- **Inbox**: Private, pressure-free conversations.
- **Me**: User boundaries and preferences.

## Design Philosophy
- Quiet, restrained, low stimulation.
- No social metrics (likes, online status, leaderboards).
- Soft typography and ample whitespace.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open the link (usually `http://localhost:5173`) in your browser.
   - The app is constrained to a mobile viewport (`max-w-md`) for the best experience on desktop.
